Fonction linéaire

Définition

Une fonction linéaire est une fonction de type $f(x)=mx$.
Le paramètre $m$ est la pente.
Une fonction linéaire passe toujours par l'origine.

Exemple

Paramètre:
$m$
Equation:
$f(x)=x$
Valeurs particulières:
zéro: $z=0$
ordonnée à l'origine: $0$

Fonction affine

Définition

Une fonction affine est une fonction de type $f(x)=mx+h$.
Le paramètre $m$ est la pente.
Le paramètre $h$ est l'ordonnée à l'origine.

Exemple

Paramètre:
$m$
$h$
Equation:
$f(x)=x+1$
Valeurs particulières:
zéro: $z=-1$
ordonnée à l'origine: $h=1$
Graphe:

Fonction quadratique

Définition

Une fonction quadratique est une fonction de type $f(x)=ax^2+bx+c$.
Le paramètre $c$ est l'ordonnée à l'origine.

Exemple

Paramètre:
$a$
$b$
$c$
Equation:
$f(x)=x^2+x+1$
Valeurs particulières:
zéro(s): pas de zéro
ordonnée à l'origine: $c=1$
Point particulier:
sommet: $(-0.5;0.75)$
